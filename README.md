# Projet – Simulation de télémétrie d’une voiture de F1

## Objectif
Concevoir une simulation de télémétrie embarquée inspirée d’une voiture de Formule 1,
en suivant une démarche d’ingénierie système.

Ce projet met l’accent sur la spécification, l’architecture, la communication et la validation
avant toute phase de développement logiciel.

## Approche
- Démarche ingénieur (pas de code immédiat)
- Versionnement progressif
- Justification des choix techniques

## Version actuelle
V1 – Définition du système (sans code)
